Explication d’internet :

Explication d’internet : world wide web -> la “toile d’araignée” Le Web est une mine d’informations incommensurable mais aussi un lieu de discussion et de débat entre personnes matures et respectueuses.
D’abord en 1969, création de l'ancêtre d’internet appelé ARPAnet.
En 1972, arrive les e-mails pour l’échange de message.
C’est finalement en 1991 que l’on a l’apparition du Web pour afficher des pages d’informations.
On peut noter qu’internet est arrivé bien avant le web, et donc que le web n’est qu’une partie d’internet. Aujourd’hui le web est dirigé par par un organisme qui est le W3C.
Cette organisme est actuellement en charge de l’évolution des technologie du web comme notamment l’Html et css
