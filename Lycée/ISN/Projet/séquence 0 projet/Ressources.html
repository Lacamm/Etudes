Dans ce Jalon nous vous demandons de rechercher l'algorithme du programme principal, il vous permettra d'établir la strucure du programme (Jalon 3)


Un programme informatique est formé de quatre types d’instructions considérées comme des petites briques de base :

• l’affectation de variables

       Variables         i, j, k : entier

x, y : réel

OK: booléen

Ch1, ch2 : chaîne de caractères

• la lecture et/ou l’écriture

• les tests
