Recherches                         2h     

     

Dans cette partie, vous devez estimer la faisabilité de votre projet. Pour celà, vous devez faire des recherches de librairies utiles pour votre projet ou bien d'autres exemples.

Afin de stocker vos recherches, vous pouvez creer un scoop-it


Vous avez la liste des modules (ainsi que le détail des classes et méthodes) de python sur le lien suivant: https://docs.python.org/3/py-modindex.html


Vous présenterez, avec précision, les fonctions que vous souhaitez utiliser dans votre projet.
